# Fleepr Design System & UX

## Visual Identity
- **Primary Color:** `#6366f1` (Indigo 500) - Represents trust and modern professionalism.
- **Background:** Soft gradients (`slate-50` to `white`) to reduce eye strain.
- **Typography:** Inter (Sans-serif) for maximum legibility.
- **Surfaces:** Glassmorphism hints for cards (subtle border, white background, soft shadow).

## UX Principles
1. **Immediate Feedback:** Every action (button click, form submission) must show a loading state.
2. **Clear Hierarchies:** The video is the hero; actions are secondary but highly visible.
3. **No Dead Ends:** Every page should have a clear path back to the main flow.

## Components
- **Buttons:** Large, rounded-xl, with subtle scale-on-hover effects.
- **Status Badges:** Pill-shaped with semi-transparent backgrounds and high-contrast text.
- **Inputs:** Minimalist borders that glow slightly on focus.
